URL2io Python SDK¶

This is the URL2io python SDK suite. Note that python2.7 is required.

This is the underlying API implementation.

How to use?¶

In [1]:
import url2io

url2io.API.server += 'demo/' # 此为演示需要，实际使用时请勿添加

Step1. Register and get token¶

In [2]:
api = url2io.API('demotoken') # 替换为你注册后得到的token
